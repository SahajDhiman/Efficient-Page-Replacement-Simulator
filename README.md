The **Efficient Page Replacement Algorithm Simulator** is a simple educational project that demonstrates how an operating system manages virtual memory using page replacement algorithms. It helps users understand how different algorithms decide which memory page to replace when a new page needs to be loaded.

This simulator implements three major algorithms:
- **FIFO (First-In-First-Out)**
- **LRU (Least Recently Used)**
- **Optimal Page Replacement**

Users can input a **reference string**, specify the **number of memory frames**, and select an **algorithm** to run. The system then simulates the process step-by-step, displaying which pages are replaced and calculating the total **page faults**, **page hits**, and **hit ratio** for each algorithm.
